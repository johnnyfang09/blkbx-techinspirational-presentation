SVG
SVG (scalable vector graphics) - an xml based vector image for for two-dimensional graphics with the support for interactivity and animation
SVG
Advantages of using SVG versus ordinary jpeg or png:
  • Scalable. Keeps that crisp clean edge without needing to increase file size.
  • Can be animated and interactive.
  • Light k-weight.
  • Supported by all modern browsers and IE9+
  • Is xml based
SVG
How it is used in code - http://jsfiddle.net/c02ocfma/1/
SVG
How to export from Illustrator:
  • Create/Open your vector graphic in Illustrator.
  • File > Scripts > SaveDocsAsSVG
  • Open the .svg file that was saved, copy and paste the xml into your code

ThreeJS
  • ThreeJS is a javascript framework that wraps boring, intense webGL code into a more usable format, just like Flash.
  • Many of the modern browsers (both on Desktop and Mobile) now support webGL.
  • ThreeJS directly taps into the device's Graphics card, offering much more processing power and very impressive performance.
  • It also is able to import models and textures in various standard industry formats.

NFC
NFC (Near-Field Communication):
  • NFC allows devices to receive data by swiping physical "tags".
  • Tags can hold a small amount of data, and can be placed almost anywhere.
NFC
Demo - http://sitefilehosting.com/nfc/
NFC
Product Examples:
  • Disney Infinity
  • ApplePay/Google Wallet
  • Parking Garage
  • Samsung S Beam
NFC

Usually used in conjunction with other apps that use NFC data as triggers: Trigger, Task Launcher, Tasker

NFC
Personal Use Examples:

"I have an NFC tag on the dashboard of my car. When I drive over to my girlfriend's house, I tap the phone to it when I'm a couple blocks away. She gets sent a text to let her know I'm almost there. We have laws where I live against texting while driving, which are sensible, and this allows me to send her a text without looking away from the road." - User
